# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - Initial release (Alpha)

### Features
- Staggered order grid algorithm for market making
- Dynamic rebalancing and partial fill handling
- Multi-bot support and PM2-based process management
- Master password & encrypted keyring for secure keys
- Standalone and manager-runner tools for dry-run testing

### Notes
- This is the first public release; use `dryRun` mode to test before using funds
- A known test assertion is outstanding in `tests/test_manager.js` expecting `active` but reporting `virtual` under some conditions
